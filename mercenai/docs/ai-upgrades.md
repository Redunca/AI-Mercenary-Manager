# AI Upgrade System

## Overview
The AI can purchase upgrades that expand capabilities, unlock new UI features, and increase automation potential.

## Upgrade Categories

### Recruit Capacity
- Increase number of recruit slots.

### Hangar Expansion
- Increase number of docking stations.
- Increase docking station size (crew capacity per ship).

### Mission Capacity
- Increase number of available missions.
- Increase number of missions that can be run in parallel.

### Mission Information
Unlock additional mission metadata:
- Estimated time to complete
- Difficulty rating
- Number of events
- Types of events

## Upgrade Mechanics
- Upgrades cost credits.
- Upgrades apply globally and persist.
- Some upgrades unlock new UI panels or daemon capabilities.
