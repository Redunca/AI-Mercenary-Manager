The list and detail subcommands (recruit, candidate, ship, shop, mission) also accept -l/--list and -d/--detail.
 
NAVIGATION
homeOpens the dashboard (overview)
split-hSplits the current panel horizontally
split-vSplits the current panel vertically
closeCloses the current panel
focus <id>Focuses panel number <id>
focus left/right/up/downDirectional navigation between panels
helpShows this help
 
RECRUITMENT
candidate listLists candidates available for recruitment
candidate detail <id>Candidate detail: stats, personality, perks/flaws
recruit hire <id>Recruits a candidate by ID
refresh(candidate-list panel) Generates new candidates
hire(candidate-detail panel) Recruits this candidate
detail <id>(candidate-list panel) Shortcut to detail view
 
RECRUITS
recruit listLists recruits with status
recruit detail <id>Recruit detail: PHY / MEN / SOC stats
rename <name>(recruit-detail panel) Renames the recruit
detail <id>(recruit-list panel) Shortcut to detail view
 
SHIPS
ship listLists ships with status (docked/on mission)
ship detail <id>Ship detail: stats, crew
ship assign <shipId> <recruitId>Assigns a recruit to a ship's crew
ship unassign <shipId> <recruitId>Removes a recruit from a ship's crew
ship rename <shipId> <name>Renames a ship
detail <id>(ship-list panel) Shortcut to detail view
assign <recruitId>(ship-detail panel) Assigns a recruit to this ship
unassign <recruitId>(ship-detail panel) Removes a recruit from this ship
rename <name>(ship-detail panel) Renames this ship
 
SHOP
shop listLists ships and consumables available for purchase
shop detail <id>Item detail: price, stats, description
shop buy <id>Buys a ship or consumable (1 unit); fails on sold-out items
walletShows the current credit balance
detail <id>(shop-list panel) Shortcut to detail view
buy <id> [quantity](shop-list panel) Buys an item by ID; blocked if sold out
buy [quantity](shop-detail panel) Buys this item; blocked if sold out
 
SELF
selfOpens the self-upgrade panel: capacity/speed upgrades bought with tokens
self buy <id>Buys the next tier of an upgrade by ID; fails if maxed or tokens are insufficient
buy <id>(self panel) Shortcut for self buy <id>
 
MISSIONS
mission listLists missions with status and progression
mission list --completedLists full mission history (success/failed), also as mission -c
mission detail <id>Mission detail: phase and progress bar
mission start <missionId> <shipId>Starts a mission with a ship
mission stop <missionId>Cancels a mission and frees up the crew
mission logs <id>Detailed logs for a mission
stop(mission-detail panel) Triggers a forced return
detail <id>(mission-list panel) Shortcut to detail view
completed(mission-list panel) Shortcut to mission history
 
LOGS
logsGlobal log: mission starts and ends
mission logs <id>Detailed logs for a mission ([SYS] / [AI] / [RECRUIT])
